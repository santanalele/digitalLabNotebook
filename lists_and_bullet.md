# Lists and bullet points

To create lists, begin a line with asterisk, or minus:

* item 1
* item 2
* item 3

This is with minus:

- item 1
- item 2
- item 3

You can make subpoints with two spaces before a point:

* item 1
  * subitem 1
  * subitem 2
* item 2

If you want to create numbered list:

1. item 1
2. item 2
3. item 3
